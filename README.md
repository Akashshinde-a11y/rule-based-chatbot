# Rule Based Chatbot 

## Overview 
A lightweight, rule-based conversational agent built in Python using Natural Language Processing (NLP) techniques and Machine Learning metrics. It matches user inputs against a predefined knowledge base using TF-IDF (Term Frequency-Inverse Document Frequency) and cosine similarity.

## Features
- Text Preprocessing
- Vectorization
- Semantic Matching
- Fallback Handling

##Technologies used
- Python
- NLTK
- Scikit-learn
